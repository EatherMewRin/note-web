第五节课

position设置

position设置可以设置标签的定位方式，常见的有:
static //默认定位
可使用margin-left/top/right/bottom设置位置
relative //相对于元素原来的位置进行偏移(不会影响其他元素布局，也就是飘逸到的位置不会占用空间)
可使用left/top/right/bottom设置位置
absolute //相对于最近的已定位祖先元素（非 static）定位
(就是在<div>或<body>或其它盒子里乱飘还不占用空间)
可使用left/top/right/bottom设置位置
fixed //相对于浏览器视口定位 导航栏就用的是这个
可用left/top/right/bottom设置位置

border-radius

border-radius可设置div的圆角 。 写法:
border-radius: 值px;

}
来设置样式

<a>标签

<a>标签用于在各个网页中跳转，写法为:

hover

hover用于设置鼠标悬停在标签上的样式，写法为:

}
